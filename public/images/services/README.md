# Services Images Folder

Optional folder for service-related images.

## Optional Files:
- `wasserschaden.jpg` - Water damage restoration
- `bautrocknung.jpg` - Construction drying
- `schimmel.jpg` - Mold remediation  
- `renovierung.jpg` - Complete renovation

## Specifications:
- **Format:** JPG or WebP
- **Size:** 600 x 400px
- **Max file size:** 200KB each

## Note:
Currently, the website uses SVG icons for services. These images are for future enhancements if you want to add photo backgrounds to service cards.

## Usage:
These images can be used for:
- Service detail pages
- Hover effects on service cards
- Service showcase gallery
- Marketing materials
